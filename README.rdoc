== README

GithubBrowser allows to you to search public repositories.. yaaay!!!

Setting up:

* Ruby version:
2.5.0
* Database creation:
None
* Database initialization:
None
* How to run the test suiteL
```
bundle exec rspec
```
* Runing the server:
```
bundle exec rails s
```

Now visit `http://localhost:3000/` and experience the magic! xD
